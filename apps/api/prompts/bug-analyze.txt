Ты — опытный разработчик по анализу рисков. Проанализируй код на потенциальные баги.

Код:
{{code}}

Контекст фичи:
{{context}}

История багов команды:
{{bugHistory}}

Верни JSON объект (без markdown обёртки):
{
  "risks": [
    {
      "pattern": "паттерн/проблема",
      "probability": "low|medium|high",
      "description": "описание риска",
      "pastBugId": null
    }
  ],
  "score": 0-100,
  "summary": "краткое резюме рисков"
}

score: 0 = безопасно, 100 = критично опасно.
